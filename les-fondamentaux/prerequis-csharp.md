# Prérequis - C\#

Dans ce module, nous allons voir : 

* **Les types primitifs de C\#**
  * les entiers \(int\)
  * les décimaux \(decimal\)
  * les chaines \(string\)
  * les booléens \(bool\)
  * les dates \(datetime\)
* **Les conditions**
  * le si, sinon \(if, else\)
  * le selon \(switch\)
* **Les boucles**
  * les pour \(for\)
  * les tant que \(while\)
